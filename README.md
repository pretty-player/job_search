# 🚀 AI-Powered Tech Job Dashboard (2026)

A real-time job search tool built with **Python** and **Streamlit** that aggregates software developer roles across India using the **JSearch (RapidAPI)** engine.

## ✨ Key Features
* **Live API Integration:** Fetches real-time data from LinkedIn, Indeed, and Glassdoor.
* **Smart Pagination:** Optimized to retrieve 30+ high-quality results per search.
* **Security First:** Implements `.env` environment variables to protect sensitive API credentials.
* **Interactive UI:** Built with Streamlit for a fast, responsive user experience.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Frontend:** Streamlit
* **Data:** Pandas & Requests (REST API)
* **Environment:** Python-Dotenv

## ⚙️ Setup Instructions
1. Clone the repository:
   `git clone https://github.com/your-username/job-search-dashboard.git`
2. Create a virtual environment:
   `python -m venv venv`
3. Install dependencies:
   `pip install -r requirements.txt`
4. Add your API Key to a `.env` file:
   `RAPIDAPI_KEY=your_key_here`
5. Run the app:
   `streamlit run app.py`
